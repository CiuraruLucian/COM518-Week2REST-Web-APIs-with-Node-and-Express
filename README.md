Overview

This project implements a REST API using Node.js and Express for managing a music database. The API allows users to search for songs, add new songs, update song information, delete songs, and purchase physical copies.

The project was also used to confirm that the Node.js development environment is correctly configured by running a simple Hello World Express server.

Technologies Used

Node.js

Express.js

SQLite Database

JavaScript / TypeScript

REST API

HTTP Methods (GET, POST, PUT, DELETE)
